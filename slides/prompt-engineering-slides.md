# เนื้อหาที่จะเรียนรู้


- Basic Prompt - การเขียน Prompt พื้นฐาน
- Zero-shot Learning - การใช้งานโมเดลโดยไม่ต้องมีตัวอย่าง

- Business Applications - การประยุกต์ใช้ในธุรกิจ

# Prompt คืออะไร?
## ความหมายและความสำคัญ

<div class="columns">
<div>

### Prompt คืออะไร?
- **คำสั่งหรือข้อความ** ที่เราใช้สื่อสารกับ AI
- **คำถามหรือคำขอ** ที่ส่งให้ AI เพื่อให้ทำงานตามที่ต้องการ
- **เหมือนการพูดคุย** กับเพื่อนที่ฉลาดมากๆ

---
### ตัวอย่าง Prompt ง่ายๆ
```
"ช่วยเขียนบทความสั้นๆ เกี่ยวกับการท่องเที่ยว"
"แปลประโยคนี้เป็นภาษาอังกฤษ"
"ช่วยคิดไอเดียสำหรับงานวันเกิด"
```

### ตัวอย่างผลลัพธ์จาก AI
**Prompt:** "ช่วยเขียนบทความสั้นๆ เกี่ยวกับการท่องเที่ยว"

**AI ตอบ:**
"การท่องเที่ยวเป็นกิจกรรมที่ช่วยให้เราได้พักผ่อนและเรียนรู้สิ่งใหม่ๆ การเดินทางไปยังสถานที่ต่างๆ ไม่เพียงแต่ช่วยให้เราได้เห็นทัศนียภาพที่สวยงาม แต่ยังช่วยให้เราได้เรียนรู้วัฒนธรรมและวิถีชีวิตของคนในท้องถิ่นนั้นๆ การท่องเที่ยวยังช่วยลดความเครียดและเพิ่มประสบการณ์ชีวิตให้กับเราได้อย่างมาก"
</div>
<div>

---

### ทำไม Prompt ถึงสำคัญ?
- **เป็นตัวเชื่อม** ระหว่างมนุษย์กับ AI
- **คุณภาพของ Prompt** = คุณภาพของผลลัพธ์
- **ยิ่งเขียนดี** = ยิ่งได้ผลลัพธ์ที่ดี

---
### เปรียบเทียบ
**Prompt ไม่ดี:** "เขียนเรื่องท่องเที่ยว"

**AI ตอบ:** "การท่องเที่ยวเป็นกิจกรรมที่สนุกและน่าสนใจ การเดินทางไปยังสถานที่ต่างๆ ทำให้เราได้เห็นสิ่งใหม่ๆ และเรียนรู้วัฒนธรรมที่แตกต่างกัน การท่องเที่ยวยังช่วยให้เราได้พักผ่อนและผ่อนคลายจากความเครียดในชีวิตประจำวัน"

---

**Prompt ดี:** "เขียนบทความ 300 คำ เกี่ยวกับการท่องเที่ยวในประเทศไทย สำหรับนักท่องเที่ยวชาวต่างชาติ"

**AI ตอบ:** "ประเทศไทย: สวรรค์ของนักท่องเที่ยวในเอเชียตะวันออกเฉียงใต้

ประเทศไทยเป็นจุดหมายปลายทางที่นักท่องเที่ยวชาวต่างชาติไม่ควรพลาด ด้วยความหลากหลายของสถานที่ท่องเที่ยว ตั้งแต่ชายหาดที่สวยงามในภาคใต้ ภูเขาสูงในภาคเหนือ ไปจนถึงเมืองหลวงที่คึกคักอย่างกรุงเทพฯ

สำหรับนักท่องเที่ยวที่ชื่นชอบธรรมชาติ เกาะภูเก็ตและเกาะสมุยในภาคใต้มีชายหาดสีขาวและน้ำทะเลใสสะอาด เหมาะสำหรับการดำน้ำดูปะการังและพักผ่อนริมชายหาด ส่วนในภาคเหนือ เมืองเชียงใหม่และเชียงรายมีภูเขาสูงและป่าเขียวขจี ที่เหมาะสำหรับการเดินป่าและชมวิว

การเดินทางในประเทศไทยสะดวกสบายด้วยระบบขนส่งสาธารณะที่ครอบคลุม และชาวไทยมีชื่อเสียงเรื่องความเป็นมิตรและต้อนรับนักท่องเที่ยว ทำให้การท่องเที่ยวในประเทศไทยเป็นประสบการณ์ที่น่าจดจำ"
</div>
</div>

# Prompt Engineering คืออะไร?
## การเขียน prompt ให้ถูกตามหลักของ engineer

<div class="columns">
<div>

### ความหมาย
- **วิธีการ** ในการเขียน Prompt ที่มีประสิทธิภาพ
- **การออกแบบคำสั่ง** ให้ AI เข้าใจและทำงานได้ตามที่ต้องการ
- **การปรับแต่งและทดสอบ** Prompt เพื่อให้ได้ผลลัพธ์ที่ดีที่สุด

---

### เป้าหมายของ Prompt Engineering
- ได้ผลลัพธ์ที่ **ถูกต้องและแม่นยำ**
- **ประหยัดเวลา** ในการปรับแก้
- **ลดความผิดพลาด** ของ AI
- **เพิ่มประสิทธิภาพ** ในการใช้งาน

---
### ตัวอย่างการปรับปรุง

**Prompt ก่อน:** "เขียนเรื่องสุขภาพ"

**AI ตอบ:** "สุขภาพเป็นสิ่งสำคัญในชีวิตของเรา การดูแลสุขภาพที่ดีจะช่วยให้เรามีชีวิตที่ยืนยาวและมีความสุข การรับประทานอาหารที่มีประโยชน์ การออกกำลังกายอย่างสม่ำเสมอ และการพักผ่อนที่เพียงพอเป็นปัจจัยสำคัญในการรักษาสุขภาพให้แข็งแรง"

---

**Prompt หลัง:** "เขียนบทความ 500 คำ เกี่ยวกับประโยชน์ของการออกกำลังกาย เน้นการลดน้ำหนัก สำหรับผู้หญิงอายุ 30-40 ปี"

**AI ตอบ:** "การออกกำลังกาย: 
การออกกำลังกายเป็นหนึ่งในวิธีที่มีประสิทธิภาพที่สุดในการลดน้ำหนักและรักษาสุขภาพสำหรับผู้หญิงในวัย 30-40 ปี ซึ่งเป็นช่วงที่ร่างกายเริ่มมีการเปลี่ยนแปลงและเผาผลาญพลังงานช้าลง

**ประโยชน์ของการออกกำลังกายต่อการลดน้ำหนัก:**

1. **เพิ่มการเผาผลาญพลังงาน:** 
2. **สร้างกล้ามเนื้อ:** การออกกำลังกายแบบ Strength 
3. **ลดความเครียด:** 

**คำแนะนำสำหรับผู้เริ่มต้น:**
เริ่มต้นด้วยการออกกำลังกายเบาๆ 15-20 นาที 3 ครั้งต่อสัปดาห์ แล้วค่อยๆ เพิ่มความเข้มข้นและระยะเวลา การออกกำลังกายอย่างสม่ำเสมอร่วมกับการควบคุมอาหารจะช่วยให้เห็นผลลัพธ์ในการลดน้ำหนักได้อย่างชัดเจน"
</div>
</div>

# Basic Prompt
## การเขียน Prompt พื้นฐาน

<div class="columns">
<div>

### หลักการพื้นฐาน
- เขียนให้ชัดเจน
- มีจุดประสงค์ที่แน่นอน
- ระบุรายละเอียดให้ครบ

---

### ตัวอย่าง Prompt ที่ดี
```
เขียนบทความเกี่ยวกับการท่องเที่ยว:
- ความยาว: 300 คำ
- กลุ่มเป้าหมาย: นักท่องเที่ยวไทย
- สไตล์: เป็นกันเอง สนุกสนาน
- เน้น: ที่เที่ยวธรรมชาติ
```

**AI ตอบ:**
"เที่ยวธรรมชาติไทย: สัมผัสความงามที่ไม่มีวันลืม

สวัสดีเพื่อนๆ นักท่องเที่ยวชาวไทย! วันนี้เรามาแนะนำที่เที่ยวธรรมชาติที่สวยงามและน่าไปสัมผัสกันนะคะ

การเที่ยวธรรมชาติไม่เพียงแต่ช่วยให้เราได้พักผ่อน แต่ยังช่วยให้เราได้เรียนรู้และเข้าใจธรรมชาติมากขึ้น แนะนำให้เตรียมอุปกรณ์ที่จำเป็น และอย่าลืมรักษาสิ่งแวดล้อมด้วยนะคะ เพื่อให้ธรรมชาติสวยงามอยู่กับเราไปนานๆ"
</div>
<div>

---

### ข้อควรระวัง
- หลีกเลี่ยงความคลุมเครือ
- ไม่ใช้คำฟุ่มเฟือย
- ระบุข้อจำกัดที่ชัดเจน

### ตัวอย่าง Prompt ที่ไม่ดี
```
เขียนเกี่ยวกับท่องเที่ยว
```

**AI ตอบ:**
"การท่องเที่ยวเป็นกิจกรรมที่คนนิยมทำกันมาก การเดินทางไปยังสถานที่ต่างๆ ทำให้เราได้เห็นสิ่งใหม่ๆ และเรียนรู้วัฒนธรรมที่แตกต่างกัน การท่องเที่ยวยังช่วยให้เราได้พักผ่อนและผ่อนคลายจากความเครียดในชีวิตประจำวัน การท่องเที่ยวมีหลายรูปแบบ เช่น การท่องเที่ยวเชิงวัฒนธรรม การท่องเที่ยวเชิงธรรมชาติ และการท่องเที่ยวเชิงผจญภัย"
</div>
</div>

# Few-shot Learning
## การใช้งานด้วยตัวอย่างเพียงเล็กน้อย

### ตัวอย่างการใช้งาน

```
แปลประโยคต่อไปนี้เป็นภาษาที่สุภาพ:

ต้นฉบับ: "เธอจะกินข้าวมั้ย"
สุภาพ: "คุณรับประทานอาหารด้วยกันไหมคะ/ครับ"

ต้นฉบับ: "ไปไหนมา"
สุภาพ: "ไปธุระที่ไหนมาหรือคะ/ครับ"

ต้นฉบับ: "อยากได้นี่"
สุภาพ: [รอ AI ตอบ]
```

### ข้อดีของ Few-shot
- AI เข้าใจรูปแบบได้ดีขึ้น
- ลดความผิดพลาด
- ได้ผลลัพธ์ที่สม่ำเสมอ

# Chain of Thought
## การคิดแบบเป็นลำดับขั้นตอน

### ตัวอย่างการแก้โจทย์คณิตศาสตร์

```
โจทย์: ร้านค้าแห่งหนึ่งลดราคาสินค้า 30% จากราคาเดิม 1,000 บาท 
หากลูกค้ามีคูปองลด 50 บาท จะต้องจ่ายเงินทั้งหมดเท่าไร?

วิธีคิด:
1. ราคาเดิม = 1,000 บาท
2. ส่วนลด 30% = 1,000 × 0.3 = 300 บาท
3. ราคาหลังลด = 1,000 - 300 = 700 บาท
4. หักคูปอง = 700 - 50 = 650 บาท

ดังนั้น ลูกค้าต้องจ่ายเงินทั้งหมด 650 บาท
```
---

### ประโยชน์
- AI แสดงกระบวนการคิด
- ตรวจสอบความถูกต้องได้
- เรียนรู้วิธีการแก้ปัญหา

# Instruction Prompt

<div>

```
1. สรุปใจความสำคัญ (3 บรรทัด)

```
</div>
<div>

---

- ใช้หัวข้อย่อย
- ระบุตัวเลขชัดเจน
- แบ่งเป็นขั้นตอน

### ข้อควรระวัง
- ไม่ซ้ำซ้อน
- ไม่กำกวม
- ไม่ขัดแย้งกันเอง

# Role-based Prompt
## การกำหนดบทบาทให้ AI

### ตัวอย่างการกำหนดบทบาท

```
คุณคือผู้เชี่ยวชาญด้านการลงทุนที่มีประสบการณ์ 20 ปี
กรุณาวิเคราะห์สถานการณ์ตลาดหุ้นไทยในปัจจุบัน โดย:

1. วิเคราะห์ปัจจัยที่ส่งผลต่อตลาด
2. ประเมินความเสี่ยง
3. ให้คำแนะนำสำหรับนักลงทุนรายย่อย
4. คาดการณ์แนวโน้มในอีก 6 เดือนข้างหน้า

ใช้ภาษาที่เข้าใจง่าย เหมาะสำหรับนักลงทุนมือใหม่
```

---
### บทบาทที่นิยมใช้
- ผู้เชี่ยวชาญด้านต่างๆ
- ครูสอน
- ที่ปรึกษาธุรกิจ
- นักวิเคราะห์

# Business Applications
## การประยุกต์ใช้ในธุรกิจ

### กรณีศึกษา: ร้านอาหาร

```
บทบาท: ผู้จัดการร้านอาหาร
งาน: สร้างแคมเปญการตลาดใหม่

Prompt:
คุณคือผู้เชี่ยวชาญด้านการตลาดร้านอาหาร
ช่วยสร้างแคมเปญการตลาดสำหรับร้านอาหารญี่ปุ่น
โดยมีเงื่อนไข:
1. ระยะเวลา: 3 เดือน
2. งบประมาณ: 50,000 บาท
3. เป้าหมาย: เพิ่มลูกค้าใหม่ 20%
4. ช่องทาง: ออนไลน์และออฟไลน์
```

---
### ประโยชน์ทางธุรกิจ
- ประหยัดเวลาและต้นทุน
- เพิ่มประสิทธิภาพการทำงาน
- สร้างสรรค์ไอเดียใหม่
- ปรับปรุงการบริการลูกค้า

### ด้านความปลอดภัย
- ระวังข้อมูลส่วนบุคคล

- ทดสอบก่อนใช้จริง

- ความโปร่งใส
- ความรับผิดชอบ
